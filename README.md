# WeirdFlex
WeirdFlex is a small programming language based on jFlex.

## Syntax

### Variables
<pre>identifier = value</pre>

### Functions
<pre>func foo<a, b>:
  #doStuff
  return bar
end</pre>

### If
<pre>if<x == b>:
  #doStuff
else
  #doStuff
end</pre>

###While
<pre>aslong<x == y>:
  #doStuff
end</pre>

## Installation
for generating the necessary files simply run "mvn clean install"
